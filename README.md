# 🚗 Machine Learning Car Price Predictor

A Machine Learning web application that predicts car prices based on user input.  
Built using **Streamlit**, this app uses a **Random Forest Regressor** with a **Pipeline** for accurate and efficient predictions.

---

## 🌐 Live Demo
https://car-price-predictor-e2rbfcubj5w6hhveksnfm4.streamlit.app/

---

## 📌 Features
- 🚀 Real-time car price prediction  
- 🤖 Random Forest Regressor model  
- 🔄 Pipeline for preprocessing + prediction  
- 📊 Interactive UI using Streamlit  
- ⚡ Fast and easy to use  

---

## 🛠️ Tech Stack
- Python  
- Streamlit  
- Pandas  
- NumPy  
- Scikit-learn

## ▶️ How to Run Locally

1. Clone the repository:
`
git clone https://github.com/your-username/car-price-predictor.git`
2. Go to the project folder:
` cd car-price-predictor `
3. Install dependencies:
` pip install -r requirements.txt `
4. Run the Streamlit app:
` streamlit run app.py `
